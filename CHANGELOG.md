# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.2.0] - 2026-02-07

### Added

- SQLite backend for fast filtering across large replay collections
- Concurrent worker-based filter pipeline
- Combo parser for character-specific combo search
- Frame-by-frame search using action states and positions
- Clip preview and video generation UI
- Batch video processing with multiple Dolphin instances

### Changed

- Major rewrite from initial prototype
- New split-panel UI with filter controls and results display
